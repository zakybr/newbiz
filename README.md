# Hurdler Website

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — The Model
- `/who-we-work-with` — Who We Work With
- `/results` — Results
- `/apply` — Apply
